## 🎯 Project Overview
The **Smart Event Dashboard** is a lightweight web application that allows users to manage their schedules. It features an intuitive interface for adding new events, viewing an organized list, and testing real-time browser interactions.

## ✨ Features
* **Event Creation:** A stylized form to input event titles, dates, categories, and descriptions.
* **Dynamic UI Updates:** Real-time DOM manipulation that replaces placeholders with formatted event cards.
* **Bulk Actions:** "Clear All" functionality to reset the dashboard and "Add Sample Events" for quick testing.
* **Interactive Demo:** A dedicated section that captures and displays real-time keyboard inputs (`keydown` events).
* **Modern Design:** A responsive, glassmorphism-inspired UI with CSS variables, gradients, and flexbox layouts.

## 🛠️ Technical Stack
* **HTML5:** Semantic structure for accessibility and SEO.
* **CSS3:** Custom properties (variables), Flexbox for layout, and transitions for smooth interactivity.
* **JavaScript (ES6+):** * `querySelector` & `getElementById` for DOM access.
    * `addEventListener` for handling clicks, form submissions, and keyboard events.
    * Template literals for dynamic HTML injection.
